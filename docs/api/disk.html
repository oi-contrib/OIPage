deleteDisk

删除文件或文件夹： 

deleteDisk(diskPath);

copyDisk

复制文件或文件夹： 

copyDisk(sourcePath, targetPath, isForce = false);

如果目标路径targetPath已经存在，会报错，如果希望成功，可以设置isForce会true。

moveDisk

始于 v1.5.0

移动文件或文件夹： 

moveDisk(sourcePath, targetPath, isForce = false);

如果目标路径targetPath已经存在，会报错，如果希望成功，可以设置isForce会true。

listDisk

始于 v1.5.0

遍历当前文件或文件夹中所有文件： 

moveDisk(sourcePath, function(fileInfo){
    // fileInfo有三个属性：name、path、folder
});

其中callback是一个函数，所有的文件都会对其进行回调。
