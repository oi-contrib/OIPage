numberFormat

数字格式化： 

let result = numberFormat(input);
属性 说明 类型 是否必输？ 默认值 备注
input 需要格式化的值 number｜string 是

dateFormat

日期格式化： 

let result = dateFormat(input, option);
属性 说明 类型 是否必输？ 默认值 备注
input 需要格式化的值 string｜Date 否 new Date()
option 表示一些约束参数 否

option

表示一些约束参数，可选，下面列举可配置项：

属性 说明 类型 是否必输？ 默认值 备注
inputFormat 输入格式 string 否 "yyyy/MM/dd" 如果你传递的是一个非正确的日期格式字符串，就可以通过这个来规范输入的格式以便程序进行解析
format 输出格式 string 否 "yyyy/MM/dd" yyyy, MM, dd, HH, mm, ss分别表示年、月、日、时、分、秒

例子 

// 2024/01/06
dateFormat(new Date())

// 04月26日 12:09:00 (2018年)
dateFormat('2018/04/26 12:09',{
    format: "MM月dd日 HH:mm:ss (yyyy年)"
})

如果字符串不满足日期规范，可以通过 inputFormat 来规范： 

// 2月14日
dateFormat('【日期】2033年2月14日',{
    format: "MM月dd日"
    inputFormat: "【日期】yyyy年MM月dd日",
})
