路由

oi.navigateTo(option)

保留当前页面，跳转到应用内的某个页面。

option参数说明

参数 类型 必填 默认值 说明
url String 是 需要跳转的应用内非 tabBar 的页面的路径

返回参数说明

返回一个Promise。

oi.redirectTo(option)

关闭当前页面，跳转到应用内的某个页面。

option参数说明

参数 类型 必填 默认值 说明
url String 是 需要跳转的应用内非 tabBar 的页面的路径

返回参数说明

返回一个Promise。

oi.reLaunch(option)

关闭所有页面，打开到应用内的某个页面。

option参数说明

参数 类型 必填 默认值 说明
url String 是 需要跳转的应用内非 tabBar 的页面的路径

返回参数说明

返回一个Promise。

oi.switchTab(option)

跳转到 tabBar 页面，并关闭其他所有非 tabBar 页面。

option参数说明

参数 类型 必填 默认值 说明
url String 是 需要跳转的应用内非 tabBar 的页面的路径

返回参数说明

返回一个Promise。

oi.navigateBack(option)

关闭当前页面，返回上一页面或多级页面。

option参数说明

参数 类型 必填 默认值 说明
delta Number 否 1 返回的页面数，如果 delta 大于现有页面数，则返回到首页

返回参数说明

返回一个Promise。
