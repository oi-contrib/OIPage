index.js

主体结构如下： 

export default function (createPage) {
    createPage({
        
        // 数据
        data:{},

        // 自定义方法
        doit(){},

        // 生命周期勾子
        onLoad(){}
    });
};

数据

定义在data中的数据，会参与页面的双向绑定，获取的时候，获取的时候使用（比如有个数据info）： 

    this.data.info

如果需要设置，应该借助 $setData 方法： 

this.$setData({
    info: "新的值"
});

自定义方法

除了生命周期勾子外，定义的方法都是自定义方法，当然，包括生命周期勾子，你都可以主动调用。

生命周期勾子

函数名 说明
onLoad 页面加载
onReady 页面加载完成
onShow 页面显示
onHide 页面隐藏
onUnload 页面被关闭
